Softwares

  • OpenAlea - Virtual Plants: since the beginning of my PhD, I am using this open source project which is primarily aimed at plant research. More specifically, I am working on the improvement of the sequence analysis library which includes an implementation of hidden Semi-Markovian models. The code is mainly written in C++ and is using Python interfaces.

  • PyENE: github organization for Eye-movement and EEGs analysis.

    • em-analysis: an overlay of Vplants Sequence Analysis for eye-movement analysis.
    • synchronized-eeg-analysis: a tool to time-lock and analyze EEGs with respect to eye-movement stimuli.
    • meta-word-embedding: an overlay of Facebook Fasttext to combine word embeddings and compute adjusted cosine similarities.
    • visu-scanpath: a simple tool for scanpath visualization
    • asynchronous-heterogeneous-hidden-semi-markov-models: a statistical model to learn and restore Asynchronous Heterogeneous Hidden Semi-Markov Models (AHHSMMs). Private ongoing development.
